# Кейс-задача № 2
### Напишите программу, которая запрашивает у пользователя ввод положительного целого числа.
#### - Реализуйте функцию, которая вычисляет факториал введенного числа.
#### - Выведите на экран результат вычисления факториала.
#### - Обеспечьте обработку возможных ошибок, таких как ввод пользователем нечисловых данных или отрицательного числа.
#### - Добавьте оптимизацию для работы с большими числами, используя библиотеку math для работы с факториалами.

Запуск программы:
```bash
python case.py
=== Вычисление факториала числа ===
Факториал числа n (обозначается n!) - это произведение всех натуральных чисел от 1 до n.

Введите положительное целое число число от 0 до 1558:
```

- Ввод отрицательного числа:
```bash
Введите положительное целое число число от 0 до 1558: -1
Ошибка: введено отрицательное число. Пожалуйста, введите положительное число.
```

- Ввод значения больше 1558 числа:
```bash
Введите положительное целое число число от 0 до 1558: 1600
Ошибка: Введенное число больше, чем 1558.
```

- Ввод тектового значниея:
```bash
=== Вычисление факториала числа ===
Факториал числа n (обозначается n!) - это произведение всех натуральных чисел от 1 до n.

Введите положительное целое число число от 0 до 1558: ASD
Ошибка: введите корректное целое число. Пожалуйста, введите целое число.
Введите положительное целое число число от 0 до 1558: 
```

- Прирывание (Ctrl + C):
```bash
=== Вычисление факториала числа ===
Факториал числа n (обозначается n!) - это произведение всех натуральных чисел от 1 до n.

Программа прервана пользователем (Ctrl+C)
```

- Прирывание (Ctrl + D):
